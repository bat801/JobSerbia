## ⚙️ Установка и запуск

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/ваш-username/jobserbia.git
cd jobserbia
```

### 2. Создайте проект в Supabase

1. Зарегистрируйтесь на [supabase.com](https://supabase.com/)
2. Нажмите **New Project** и заполните данные
3. Дождитесь создания базы данных (1–2 минуты)

### 3. Создайте таблицу `vacancies`

Перейдите в раздел **SQL Editor** и выполните этот скрипт:

```sql
-- Создаём таблицу вакансий
CREATE TABLE vacancies (
    id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
    title text NOT NULL,
    company text NOT NULL,
    city text NOT NULL,
    salary_min integer,
    salary_max integer,
    contacts text NOT NULL,
    description text NOT NULL,
    is_active boolean DEFAULT true,
    created_at timestamp with time zone DEFAULT now()
);

-- Отключаем RLS для простоты (для прототипа)
ALTER TABLE vacancies DISABLE ROW LEVEL SECURITY;
```

### 4. Вставьте тестовые данные (по желанию)

```sql

VALUES

('Менеджер по продажам', 'Balkan Sales Group', 'Нови-Сад', 1000, 2000, 'balkan.sales@example.com', 'Открыта вакансия менеджера по продажам. Знание русского и английского/сербского обязательно. Оклад + % с продаж.'),
('Бариста', 'Coffee NS', 'Нови-Сад', 600, 800, '+381601234567', 'Ищем бариста в новую кофейню. Опыт работы приветствуется. График 2/2.'),
('Системный администратор', 'IT Solutions', 'Белград', 1500, 2500, 'Telegram: @sysadmin_hr', 'Поддержка офисной инфраструктуры, настройка сетей. Требуется понимание Linux и Windows Server.');
```

### 5. Подключите Supabase к проекту

1. Перейдите в **Project Settings → API** в Supabase
2. Скопируйте **Project URL** и **anon public** ключ
3. Откройте файл `app.js` и замените значения:

```javascript
const SUPABASE_URL = 'https://ваш-проект.supabase.co';
const SUPABASE_KEY = 'ваш-anon-публичный-ключ';
```

### 6. Запустите сайт локально

Просто откройте `index.html` в браузере.  
Либо используйте Live Server в VS Code:

```bash
# Установите расширение Live Server и нажмите "Go Live"
```

### 7. Задеплойте на GitHub Pages

1. Создайте репозиторий на GitHub
2. Залейте файлы:

```bash
git add .
git commit -m "Initial commit"
git push origin main
```

3. Перейдите в **Settings → Pages** вашего репозитория
4. В разделе **Branch** выберите `main` и папку `/ (root)`
5. Нажмите **Save**
6. Через 1–2 минуты сайт будет доступен по адресу:  
   `https://ваш-username.github.io/jobserbia/`

## ❓ Частые вопросы

**Как сменить пароль админки?**  
Откройте `admin.html`, найдите строку:
```javascript
const ADMIN_PASSWORD = 'admin123';
```
Замените на свой пароль.

**Почему не работают фильтры?**  
Убедитесь, что вы правильно вставили ключи Supabase в `app.js`.  
Проверьте консоль браузера (F12) на наличие ошибок.

**Как скрыть вакансию, не удаляя её?**  
В админ-панели при редактировании снимите галочку **«Активна»**.  
Вакансия перестанет отображаться на главной странице.

**Можно ли загружать фото/логотипы?**  
В текущей версии нет. Это будет добавлено в следующих релизах.
